# Kupola UI Integration Example

This example shows how a Kupola 2.x app can wrap PIVOT with Drawer, Modal, Table, and Message primitives.

It covers:

- capability catalog in a Drawer
- command preview in a Drawer
- command confirmation in a Modal
- plan graph preview in a Drawer
- plan approval in a Modal

Run:

```bash
npm run example:kupola-ui-integration
```

The Kupola primitives are mocked here so the integration shape stays runnable inside this repository.
