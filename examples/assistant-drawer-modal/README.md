# Drawer / Modal Integration Example

This example shows how a host app can wire PIVOT trusted UI hooks to Drawer and Modal surfaces.

It covers:

- command preview in a Drawer
- command confirmation in a Modal
- plan preview in a Drawer
- plan approval in a Modal

Run:

```bash
npm run example:assistant-drawer-modal
```

In a real Kupola 2.x app, replace the mocked `drawer` and `modal` objects with the project's own Drawer and Modal primitives.
